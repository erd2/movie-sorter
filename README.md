# Movie Genre Sorter

A simple Flutter application that displays a list of movies and allows users to filter them by genre.

## Project Idea
The goal of this project is to demonstrate basic Flutter concepts such as:
*   Displaying a list of items using `ListView`.
*   Using `Card` and `ListTile` widgets for UI.
*   State management using `setState` to filter data.
*   Using a `DropdownButton` for user input.
*   Organizing code into models, screens, and widgets.

## Features
*   View a list of movies with their title, genre, and release year.
*   Filter the list by selecting a genre (Action, Comedy, Drama, Horror, or All).
*   Clean and modern Material Design UI.

## How to run the project

1.  **Prerequisites:**
    *   Flutter SDK installed on your machine.
    *   An Android/iOS emulator or a physical device connected.

2.  **Installation:**
    ```bash
    git clone https://github.com/erd2/movie-sorter.git
    cd movie-sorter
    ```

3.  **Run the app:**
    ```bash
    flutter run
    ```
